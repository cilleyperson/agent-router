# Security Policy

## Supported Versions

We actively maintain and provide security updates for the following versions of the AI Agent Router Proxy:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

If you discover a potential security vulnerability in this project, please report it privately to help protect our users. You can report vulnerabilities by:

1.  Opening a private vulnerability report directly through GitHub:
    *   Navigate to the **Security** tab of this repository.
    *   Under **Vulnerability reporting**, click **Report a vulnerability**.
2.  Or emailing the maintainer privately at `security@cilleyperson.dev`.

Please include the following details in your report:
*   A detailed description of the vulnerability and its potential impact.
*   Step-by-step instructions (or a proof-of-concept script) to reproduce the vulnerability.
*   Your environment details (OS, Python version, configuration settings).

### Our Response Timeline
*   **Acknowledgment**: We will acknowledge receipt of your report within **48 hours**.
*   **Assessment & Triage**: We will investigate and confirm the vulnerability details.
*   **Fix & Disclosure**: We aim to release a patch and publish a security advisory within **30 days** of triage.
